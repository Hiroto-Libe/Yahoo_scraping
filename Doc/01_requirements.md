# 要件定義書（Yahooショッピング 商品検索・価格監視）

## 1. 目的
- 価格監視と商品一覧の収集を行う

## 2. 対象範囲
- 対象サービス: Yahooショッピング
- 取得方法: Yahoo Shopping API（ItemSearch）
- 参照先: https://developer.yahoo.co.jp/webapi/shopping/v3/itemsearch.html

## 3. 機能要件
### 3.1 入力
- カテゴリは選択可能
- キーワードは入力形式で指定

### 3.2 出力
- 出力形式: CSV

### 3.3 取得項目
- 価格監視に必要な項目
- 商品一覧の収集に必要な項目

## 4. 非機能要件
### 4.1 実行環境
- Python
- HTML/CSS
- ローカルサーバ

### 4.2 取得件数/頻度
- 1回あたりの件数上限: 50件
- 実行頻度: 手動（不定）

### 4.3 運用・エラー対応
- 失敗時のリトライ: 最大2回
- ログ出力あり

### 4.4 認証/秘密情報
- アプリIDの管理方法は特に指定なし

### 4.5 仕様外の制約
- 現時点で制約なし

## 5. 未確定事項・今後の検討
- CSVに出力する具体的な項目の確定
- カテゴリ選択のUI/入力方式の詳細
- ログの出力形式と保存先
